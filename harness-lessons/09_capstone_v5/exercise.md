v5 在 8k 下峰值仅 837——余量巨大。把窗口压到多小它才断？

```python
import tempfile
from research_assistant import steering
from eval_agent.harness_runs import run_full_longhaul
for limit in (8000, 4000, 2000, 1200):
    steering.set_db_path_for_test(tempfile.mktemp(suffix=".db"))
    try:
        r = run_full_longhaul(workspace_base=tempfile.mkdtemp(),
                              memory_base=tempfile.mkdtemp())
        # 注：run_full_longhaul 的主窗 limit 目前固定 8k——给它加 window_limit
        # 参数透传 main_ledger 是本题第一步

        print(limit, "断了：", type(e).__name__)
```
